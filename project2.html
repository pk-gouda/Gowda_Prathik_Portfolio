Project Description

I employed a range of deep learning and machine learning techniques, such as Long Short-Term Memory (LSTM) networks, Elastic Net regression, and Random Forest algorithms, to accurately estimate stock prices.

The backend was developed using Python, ensuring robust data processing and model integration. I hosted the application on AWS EC2, leveraging its scalability and reliability. For efficient database management, I utilized Amazon RDS, which streamlined data storage and retrieval. This combination of advanced techniques and cloud-based infrastructure facilitated precise stock price predictions and seamless user experience.

Technologies Used

  • Programming Language: Python
  • Python Libraries: Numpy, Scipy and Pandas.
  • Machine Learning Algorithms: Long Short-Term Memory (LSTM), Elastic Net Regression and Random Forest.
